# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Release Pilot, please report it
responsibly via [GitHub's private vulnerability reporting](https://github.com/YOUR_USERNAME/release-pilot/security/advisories/new).

**Please do NOT open a public issue for security vulnerabilities.**

## Supported Versions

| Version | Supported |
|---------|-----------|
| Latest  | ✅        |
| < Latest | ❌       |
